Programación en C

Introducción

El lenguaje de programación C es uno de los lenguajes de programación más utilizados en la industria. Fue desarrollado en los años 70 y es muy conocido por su eficiencia y simplicidad.

Tipos de Datos

  • int: Números enteros.
  • float: Números de punto flotante.
  • double: Números de doble precisión.
  • char: Caracteres individuales.

Estructuras de Control

Estructura Selectiva

Las estructuras selectivas permiten tomar decisiones basadas en condiciones.

If-else

if (condición) {
    // Código si la condición es verdadera
} else {
    // Código si la condición es falsa
}

Switch

switch (variable) {
    case valor1:
        // Código para valor1
        break;
    case valor2:
        // Código para valor2
        break;
    default:
        // Código si no coincide con ningún caso
}

Estructura Repetitiva

Las estructuras repetitivas permiten ejecutar un bloque de código varias veces.

For

for (inicialización; condición; actualización) {
    // Código a ejecutar
}

While

while (condición) {
    // Código a ejecutar mientras la condición es verdadera
}

Do-while

do {
    // Código a ejecutar
} while (condición);

Arreglos

Los arreglos en C son estructuras que almacenan múltiples valores del mismo tipo en una sola variable.

int numeros[5] = {1, 2, 3, 4, 5};

Métodos de Ordenamiento

Ordenamiento por Burbuja

void bubbleSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        for (int j = 0; j < n-i-1; j++) {
            if (arr[j] > arr[j+1]) {
                // Intercambiar
                int temp = arr[j];
                arr[j] = arr[j+1];
                arr[j+1] = temp;
            }
        }
    }
}

Ordenamiento por Selección

void selectionSort(int arr[], int n) {
    for (int i = 0; i < n-1; i++) {
        int minIdx = i;
        for (int j = i+1; j < n; j++) {
            if (arr[j] < arr[minIdx]) {
                minIdx = j;
            }
        }
        // Intercambiar
        int temp = arr[minIdx];
        arr[minIdx] = arr[i];
        arr[i] = temp;
    }
}

Ordenamiento por Inserción

void insertionSort(int arr[], int n) {
    for (int i = 1; i < n; i++) {
        int key = arr[i];
        int j = i - 1;
        while (j >= 0 && arr[j] > key) {
            arr[j + 1] = arr[j];
            j = j - 1;
        }
        arr[j + 1] = key;
    }
}
